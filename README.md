# Deepfake Detection: An Artifact Based Approach

**Group 9 - CS4366-001 - Texas Tech University**

Team page for our Senior Capstone project on Deepfake Detection using artifact-based analysis techniques.

## Project Overview

This project focuses on developing a robust deepfake detection system by identifying subtle inconsistencies and artifacts introduced during the deepfake generation process.

## Local Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Deployment on Railway

1. Push this repository to GitHub
2. Go to [Railway](https://railway.app)
3. Create a new project and select "Deploy from GitHub repo"
4. Select this repository
5. Railway will automatically detect the configuration and deploy

### Custom Domain Setup

1. In your Railway project, go to Settings → Domains
2. Add your custom domain (e.g., `deepfake-group9.com`)
3. Add the provided DNS records to your domain registrar
4. Wait for DNS propagation (can take up to 48 hours)

## Updating Content

To update team members, meetings, or timeline, edit the data in `src/App.jsx`:

- `teamMembers` - Array of team member objects
- `meetings` - Array of meeting schedule objects
- `milestones` - Array of project milestone objects
